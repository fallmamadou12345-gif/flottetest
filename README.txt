=====================================================
  SyNdongo - Guide ultra-simple
  (Version Node.js - sans Python)
=====================================================

PRÉREQUIS : Node.js uniquement
  → https://nodejs.org
  → Cliquez sur le bouton vert "LTS"
  → Installez normalement

DÉMARRER :
  → Double-cliquez start.bat
  → Ouvrez http://localhost:8000

VOS DONNÉES :
  → Sauvegardées dans syndongo_data.json
  → Ce fichier se crée automatiquement
